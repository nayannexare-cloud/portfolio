## Running Locally

No setup needed. Just open index.html in any browser.

---

## Deploying to GitHub Pages

1. Push this repo to GitHub
2. Go to Settings → Pages
3. Set source to Deploy from branch → main → root
4. Save — live in about 60 seconds
